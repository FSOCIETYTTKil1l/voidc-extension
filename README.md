# VoidC~ Examples

These examples demonstrate the features of VoidC~.


``` ##Running the Examples
"Bash

## Compile an example
voidc 01_hello_world.c~

## Run
./01_hello_world.exe
"

## Example Overview

1. **01_hello_world.c~** - Simplest Program

2. **02_variables.c~** - Variables and Arithmetic
3. **03_if_else.c~** - Conditions and Branching
4. **04_loops.c~** - While Loops
5. **05_functions.c~** - Functions with Return Values
6. **06_fibonacci.c~** - Fibonacci Algorithm
7. **07_pointers.c~** - Pointer Usage
8. **08_prime_checker.c~** - Prime Number Testing
9. **09_factorial.c~** - Recursive Factorial
10. **10_password_checker.c~** - A Practical Tool

## Learning Path

Beginner: 1 → 2 → 3 → 4 → 5
Intermediate: 6 → 7 → 8 → 9
Practical: 10 + Your Own Projects!
